# PARA-METER-V2.1-ESP
PARA-METER for ESP32

Code adapted to be used with ESP32

* No need for PROGMEM
* Uses HardwareSerial (Serial2 with pins RX 16 and TX 17)
* Uses open pins 35 and 34 to read random data
* Adopted for ADC1
